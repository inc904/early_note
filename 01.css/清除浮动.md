# 清除浮动四种方法

## 0、why？

当父元素不给高度的时候，

- 内部元素不浮动的时候会撑开，而浮动的时候，会变成一条线，失去高度。

## 1、添加额外元素

在最后一个浮动的元素后面，添加一个标签，给他设置`clear: both;`

无意义标签，语义化差。

## 2、给父元素添加 overflow 属性

触发了 BFC ，实现清除浮动。

内容多的时候，无法显示要溢出的元素。

## 3、使用 after 伪元素

```css

.clearfix{
 	content: '';
    height: 0;
    visibility: hidden;
    clear: both;
    display: block;
    
}
.clearfix{
    *zoom: 1; /* 兼容ie*/
}
```



## 4、使用after和before双伪元素

```css
  .clearfix:after,.clearfix:before{
        content: "";
        display: table;
    }
    .clearfix:after{
        clear: both;
    }
    .clearfix{
        *zoom: 1;
    }
```
