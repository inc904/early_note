# Bootstrap -> css框架__twitter 出品

## 一、版本对比

- bootstrap3：

  - 兼容 IE9+
  - less 编写

- bootstrap4：

  + 兼容IE10+
  + 使用 flexbox 布局
  + 抛弃 Nomalize.css
  + 提供布局和reboot版本：相当于提供了三个版本：
    * 1.全部版本。
    * 2.只有布局的版本。
    * 3.reboot版本
  + sass 编写
