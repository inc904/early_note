## 获取文本节点内容

```html
<div id="demo">只获取我<a href="">别管我</a> </div>
```

1. jquery获取

```js
var a = $('#demo').prop('firstChiild').nodeValue
```

2. js原生获取

```js
var a = document.getElementById('demo').firstChild.nodeValue
```
