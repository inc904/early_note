## js的数据类型

### 基本数据类型

string、Boolean、number、undefined、object、null

### 引用数据类型

Object（array、Date、RegExp，Function）

## typeof  返回的数据类型

- object
- number
- function
- boolean
- string
- undefined

## split() ) join() 的区别

```js
'hello'.split('') //  ["h", "e", "l", "l", "o"]
["h", "e", "l", "l", "o"].join('-') // "h-e-l-l-o"
```
