## 清除浮动：
```css
//单伪元素，代表网站： 百度、淘宝网、网易等

 .clearfix:after {
    content: ".";
    display: block;
    height: 0;
    clear: both;
    visibility: hidden;
 }   
 .clearfix {*zoom: 1;}   /* IE6、7 专有 */

 //双伪元素，代表网站： 小米、腾讯等
 .clearfix:before,.clearfix:after { 
  content:"";
  display:table; 
}

.clearfix:after {
 clear:both;
}
.clearfix {
  *zoom:1;
}
```
